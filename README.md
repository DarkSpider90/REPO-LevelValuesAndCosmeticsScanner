<h3 align="center">A small HUD helper for loot value and cosmetic boxes</h3>
<br>

## About

Level Values And Cosmetics Scanner is a simple quality-of-life mod for R.E.P.O.

It keeps a small counter on your HUD so you can quickly see how much valuable loot is still left outside the extraction zone and how many items are still out there.

It also shows cosmetic boxes by rarity. If there are cosmetic boxes on the level, colored square icons appear above the value counter with the amount of boxes for each rarity.

## What It Shows

* Total value of valuables not yet brought to extraction.
* Number of valuable items still outside extraction.
* Cosmetic box counters for Common, Uncommon, Rare, and Ultra Rare boxes.
* A compact bottom-right HUD that can be always visible or shown only with the map.

## Configuration

The mod includes a simple config file where you can:

* Toggle the value counter
* Toggle item count
* Toggle cosmetic box tracking
* Enable or disable always-on mode
* Adjust HUD position

## Installation

### Mod Manager

Install it with a Thunderstore-compatible mod manager such as r2modman or Thunderstore Mod Manager.

### Manual

1. Install BepInEx for R.E.P.O. if you do not already have it.
2. Download the mod package.
3. Open your game folder.
4. Put `LevelValuesAndCosmeticsScanner.dll` into:

```text
BepInEx/plugins/LevelValuesAndCosmeticsScanner/
```
