## 1.0.0

* Initial release of Level Values And Cosmetics Scanner.
* Tracks the total value and item count of valuables outside the extraction zone.
* Adds cosmetic box rarity counters for Common, Uncommon, Rare, and Ultra Rare boxes.
* Displays custom runtime-drawn rarity icons with the amount of boxes for each rarity.
* Includes config options for value, item count, cosmetic box scanning, visibility, and UI position.
